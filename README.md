## COMO USAR NO CELULAR

1. Abra o **Chrome** ou **Safari**
2. Acesse:  
   `https://seuusuario.github.io/ar-digital`
3. Preencha o recibo
4. Assine com o dedo
5. Tire fotos (opcional)
6. Clique em **"GERAR PDF"**
7. O PDF é baixado automaticamente

> **NUNCA use `file://` ou WhatsApp** → bloqueia câmera
